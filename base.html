Les boucles

Une boucle est une structure qui permet de repeter un bloc de code tant qu'une condition est vraie.

C'est une façon automatique de repeterdes instructions sans devoir les ecdrire plusieurs fois

Les Types de boucles

  • La boucle While
  • La boucle Do...While
  • La boucle For
  • La boucle For...of
  • La boucle For...in

La Boucle While

On l'utilise quand on ne connais pas forcément à l'avance le nombre de tours

            
                // condition => Cette condition doit être vraie pour pouvoir continuer la boucle (comme le if, cependant ne jamais mettre de == ou de ===)
                While (condition) {
                    // code à exécuter
                }

Exemple concret

            
                let n = 0;
                while (n < 6) {
                    console.log(n + " * 5 est égal à " + n * 5);
                    n++
                }

La boucle Do...While

C'est comme la boucle while, mais elle s'execute au moins une fois avant de tester la condition

            
              do {
                //
              } while (condition);

Exemple concret

            
                let des;
                do {
                des = Math.floor(Math.random()* 7)
                 console.log(des)
                } while (des !== 6)
                console.log("Tu à trouver le dés magique!");

La boucle for

La plus classique et la plus utiliser, quand on connait à l'avance le nombre de repetition

            
             
                for (initialisation, condition, incrémentation)

Exemple concret

            
             
                for (let i = 0; i < 5; i++){
                    console.log("Compteur : " + i);
                }
