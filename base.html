Les boucles

Une boucle est une structure qui permet de repeter un bloc de code tant qu'une condition est vraie.

C'est une façon automatique de repeterdes instructions sans devoir les ecdrire plusieurs fois

Les Types de boucles

  • La boucle While
  • La boucle Do...While
  • La boucle For
  • La boucle For...of
  • La boucle For...in

La Boucle While

On l'utilise quand on ne connais pas forcément à l'avance le nombre de tours

            
                // condition => Cette condition doit être vraie pour pouvoir continuer la boucle (comme le if, cependant ne jamais mettre de == ou de ===)
                While (condition) {
                    // code à exécuter
                }

Exemple concret

            
                let n = 0;
                while (n < 6) {
                    console.log(n + " * 5 est égal à " + n * 5);
                    n++
                }

La boucle Do...While

C'est comme la boucle while, mais elle s'execute au moins une fois avant de tester la condition

            
              do {
                //
              } while (condition);

Exemple concret

            
                let des;
                do {
                des = Math.floor(Math.random()* 7)
                 console.log(des)
                } while (des !== 6)
                console.log("Tu à trouver le dés magique!");

La boucle for...in

Permet de parcourir les propriétés d'un objet

Les mots clées BREAK et CONTINUE

BREAK permet de sortir immediatment d'une boucle

            
          for (let i = 0; i < 10; i++){
    if ( i === 5) {
        break;
    }
    console.log(i)
}

Le CONTINUE permet de sauter une itération sans arreter la boucle

            
             
              for (let i = 0; i < 10; i++){
  if ( i % 2 === 0){
    continue;
  }
  console.log(i)
}

Itérables & Itérateurs

Un itérable est un objet qui peut etre parcourus(itérere) par un itérateur comme une boucle

Exemple d'objet itérables

  • Tableaux []
  • String "abc"
  • Arguments d'une fonction
  • Les clés dans les objets

Les méthodes itératrices

Listes des méthodes itératrice

  • .forEach()
  • .map()
  • .filter()
  • .reduce()
  • .find()
  • .some()
  • .every()

Aucune de ces méthodes ne modiefie les tableaux d'origines

const videoGames =['Mario', "Pandemoniom", "Gran Turismo", "Metal Gear", "Zelda", "Hollow Knight", "Rocket League"]
