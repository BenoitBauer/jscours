1. Introduction à JavaScript

C'est quoi le JavaScript ?

Le JavaScript est un language de programmation qui donne de l'intératactivité aux pages web

  • HTML = structureou le squelette
  • CSS = style (les couleurs), la mise en page ...
  • JS = logique (les reaction à un utilisateur, des calculs et la dynamique de la page)

Où s'éxecute le JS ?

  • Dans le navigateur (all)
  • Sur un serveur qui tourne avec Node.je -> il ne sera plus un language front end mais back end

A retenir : même language mais deux environnement differents

Premier pas : console, balise script

  1. Ouvrir votre navigateur préféré
  2. Clique droit -> inspecter ou si vous êtes sur windows F12.
  3. Aller dans l'onglet console
