SetTimeOut () et SetInterval()

SetTimeOut()

Ca execute une fonction une seule fois apres un certain temps

            
         
                setTimeout(function(){
                     console.log("3 secondes se sont écoulées !")
                }, 3000)

SetInterval()

execute une fonction de manière repetée à intervalle regulier

            
         
               setInterval(function(){
                    console.log("Une seconde viens de passées !")
                }, 1000)

clearTimeout() et clearInterval()

POur pouvoir supprimer vos timers, il faudra les mettre dans une variable

            
         
             
                const timer = setTimeout(function(){
    console.log("3 secondes se sont écoulées !")
}, 3000)

clearTimeout(timer) //c'est plus interessant dans un événement

const interval = setInterval(function(){
    console.log("3 secondes se sont écoulées !")
}, 3000)

clearInterval(interval) //c'est plus interessant dans un événement
